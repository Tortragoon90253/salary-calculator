🔥 ตั้งค่า Firebase Firestore
เชื่อมต่อกับ Firebase Firestore เพื่อบันทึกข้อมูลบน Cloud
เข้าถึงได้จากทุกเครื่องและทุก Browser
📋 วิธีตั้งค่าครั้งแรก (ทำครั้งเดียว)
  1. ไปที่ console.firebase.google.com แล้วสร้างโปรเจกต์ใหม่
  2. เปิด Firestore Database → เลือก mode Test mode
  3. ไปที่ Project Settings → General แล้ว Copy
      • Web API Key และ Project ID
  4. กรอกข้อมูลด้านล่าง แล้วกด บันทึก Config
Document ID ใช้แยกข้อมูลระหว่างผู้ใช้ — ใส่ชื่อใดก็ได้ (เช่น ชื่อของคุณ)
☁️ ตั้งค่า Google Sheets
เชื่อมต่อกับ Google Sheets เพื่อบันทึกข้อมูลบน Cloud
เข้าถึงได้จากทุกเครื่องและทุก Browser
📋 วิธีตั้งค่าครั้งแรก (ทำครั้งเดียว)
  1. สร้าง Google Spreadsheet ใหม่ (ไม่ต้องสร้าง Sheet เอง ระบบสร้างให้อัตโนมัติ)
  2. คลิก Extensions → Apps Script
  3. วาง code จากไฟล์ AppsScript_Code.js ที่อยู่ในโฟลเดอร์เดียวกัน แล้ว Save
  4. คลิก Deploy → New Deployment → Web App
      • Execute as: Me
      • Who has access: Anyone
  5. Copy Deployment URL (ขึ้นต้นด้วย script.google.com) แล้ววางด้านล่าง
  6. กด บันทึก URL — ระบบจะโหลดข้อมูลจาก Sheets ทันที
URL นี้เก็บไว้ใน Browser นี้เท่านั้น (localStorage) — ต้องกรอกใหม่ถ้าเปลี่ยน Browser
